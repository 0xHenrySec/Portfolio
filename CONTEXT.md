# Portfolio de Cibersegurança

## Objetivo
Desenvolver um portfólio profissional de cibersegurança utilizando GitHub Pages, com foco em apresentar projetos práticos, laboratórios e experiências técnicas.

O portfólio deve ser direto, técnico e focado em demonstrar habilidades reais, evitando excesso de conteúdo acadêmico ou certificações.

---

## Stack do Projeto
- Astro
- TailwindCSS
- GitHub Pages (deploy via GitHub Actions)

---

## Estrutura do Site

O site deve ser uma homepage principal com seções organizadas e páginas individuais para cada projeto.

### Seções da Home
- Hero (apresentação principal)
- Sobre
- Projetos (principal seção)
- Medium (artigos técnicos)
- Contato

## Requisitos para cada projeto

Cada projeto deve ter sua própria página contendo:

- Visão geral
- Objetivo
- Tecnologias utilizadas
- Arquitetura ou funcionamento
- Implementação
- Resultados / evidências
- Aprendizados
- Links (GitHub, artigos, etc.)

---

## Design e UI

- Tema escuro (dark mode)
- Visual limpo e profissional
- Estilo voltado para cibersegurança
- Layout responsivo
- Uso de cards para projetos

---

## Navegação

Menu simples:

- Home
- Projetos (ou scroll automático)
- Medium
- Contato

## Diretrizes importantes

- Foco em experiência prática (hands-on)
- Evitar foco em certificações
- Evitar textos genéricos
- Priorizar clareza técnica
- Conteúdo em português
- Termos técnicos podem permanecer em inglês

---

## Objetivo final

Criar um portfólio que:

- Demonstre capacidade técnica real
- Destaque projetos e laboratórios
- Seja visualmente profissional
- Seja simples de navegar
- Seja relevante para recrutadores de cibersegurança
